Description

Dapper Dappled Pacific - Santa Cruz Mountains. We're trying to replicate this atmospheric landscape. In this experiment, I used Glitch to recreate this landscape, starting with the remix of A. Smith's living impressions glitch page.

Technical

I used the color picker tool in Photopea and assembled a palette of colors. Then, I started with simple rectangles to fill in the sky, clouds, and landscape horizone. I used a noise function to create the ridgelines, for the hills in the background. Lastly, for the trees, I imported the triangle creation code from A. Smith's living impressions page.

Reflection

Working on this experiment, I realized how much colors do to bring the impression together. Using accurate colors using a color picker helps bring the image together, as long as the general shapes and lines are in place.
